Frontend Mentor Challenges

This website is a collection of challenges that I have completed from frontendmentor.com. These challenges were super fun and really advanced my skills in HTML, CSS and Javascript. Click the links in the boxes to see live solutions to the challenges.

1/5

Social Media Dashboard

For this challenge I had to recreate several cards with social media analytics and create a dark mode toggle. I was able to recreate this utilizing CSS grid without any media queries. The dark mode toggle was made by storing colors in CSS variables and using Javascript to toggle the dark mode class on and off. As a bonus, I learned to use local storage to save the user's preference on future visits.
